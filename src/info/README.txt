Для запуска приложения необходимо запустить файл Server.java и далее необходимое количество Клиентов файлом Client.java

GUI
Для начала работы с окном парковки необходимо выбрать положение въезда.

Для занести машину на парковку необходимо нажать на соответствующую кнопку (грузовой или легковой автомобиль)

Для удаления машины с парковки необходимо навести на машину и нажать на кнопку удаления ("Х")




Консоль
В случае удачного подключения клиент запросит ввести положения входа:
	Input entry point (top/bot):
	Необходимо ввести "top" или "bot"

Далее выведется меню для работы с программой: 
	Enter number:
	1 - Car coming
	2 - Car is leaving
	3 - Show parking
	4 - Exit

Чтобы ввести прибывшую машину необходимо ввести ее размер, чтобы проверить есть ли для нее место
	2-легковой автомобиль
	3-грузовой автомобиль
	
Чтобы зарегистрировать отъезд автомобиля необходимо ввести место с которого он уезжает(название зоны и номер без пробела)
	Например: A0, C20

Формат отображения парковки:
	Area A: 
 	|A0 2(1)|  |A2 3(0)|
	
	где: А0/А2 - название зоны и номер места
	     2/3 - размер места
	     1/0 - статус места. 1 - занято, 0 - свободно.
